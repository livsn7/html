link para site externo

link externo aponta para um endereço URL fora do seu proprio site ou do servidor onde os seus arquivos estao hospedados.

como que ele fuciona?

para que o navegador saiba que deve sair do seu dominio, o atributo 'HREF' precisa do caminho absoluto, ou seja, o endereço completo començando pelo protocolo (http:// ou https://). site para ouvir musicas

0 Atributo target="blank" força o link a abrir em uma nova aba do navegador, garantindo que o usuario do site nao feche o seu site por engano ao clicar no link
